# Battlecode

## Setup

```bash
uv sync
source .venv/bin/activate
```

## Commands

### Run a match

```bash
cambc run starter starter default_small1.map26
```

### Watch replay

```bash
cambc watch replay.replay26
```

### Edit your bot

```bash
bots/starter/main.py
```

### Submit

```bash
cambc submit bots/starter
```
